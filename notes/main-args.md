# `main`의 두 인자

C++에서 `main` 함수는 보통 이렇게 시작합니다.

```cpp
int main(int argc, char **argv)
```

여기서 `int`와 `char **`는 `main`의 매개변수 종류를 뜻합니다.

- `int argc`
  - 프로그램이 실행될 때 전달된 인자의 개수입니다.
  - 쉽게 말해, "입력값이 몇 개 들어왔는지" 세는 숫자입니다.
  - 보통 프로그램 이름도 포함해서 셉니다.

- `char **argv`
  - 전달된 인자들의 문자열 목록입니다.
  - `argv[0]`은 보통 프로그램 이름이고, `argv[1]`, `argv[2]`처럼 뒤에 실제 인자들이 들어갑니다.
  - 쉽게 말해, "입력값들이 줄지어 들어 있는 문자열 상자"라고 보면 됩니다.

## 예시

```bash
./main file.voy
```

이때는 보통:

- `argc == 2`
- `argv[0] == "./main"`
- `argv[1] == "file.voy"`
