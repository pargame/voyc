# `fstream` 노트

`fstream`은 파일을 읽고 쓰는 기능을 제공하는 C++ 표준 라이브러리의 한 부분입니다.

## `std::ifstream`

`std::ifstream`은 **입력용 파일 스트림**입니다.
즉, 파일에서 내용을 읽을 때 사용합니다.

예를 들어 다음 코드처럼 씁니다.

```cpp
std::ifstream file(argv[1]);
```

여기서 의미는 다음과 같습니다.

- `std::ifstream` : 파일을 읽기 위한 객체
- `file` : 그 객체의 이름
- `argv[1]` : 열고 싶은 파일 경로

쉽게 말하면, `ifstream`은 파일을 "읽어오는 통로"입니다.

## `file.is_open()`

`is_open()`은 파일이 실제로 열렸는지 확인하는 함수입니다.

```cpp
if (!file.is_open()) {
  // 파일 열기 실패
}
```

이 코드는 파일이 안 열렸을 때를 확인합니다.

- `true`면 파일이 열려 있는 상태
- `false`면 파일이 열리지 않은 상태

쉽게 말하면, 문을 열려고 시도한 뒤 정말 열렸는지 확인하는 손잡이 같은 역할입니다.

## 왜 필요한가

파일 경로가 틀리거나, 파일이 없거나, 권한이 없으면 파일을 못 엽니다.
그래서 `is_open()`으로 확인한 뒤 안전하게 처리합니다.

## 현재 소스코드와 연결

현재 소스에서는 이런 흐름입니다.

1. 실행 인자로 파일 경로를 받음
2. `std::ifstream file(argv[1]);`로 파일을 열어 봄
3. `file.is_open()`으로 성공 여부를 확인함
4. 실패하면 에러 메시지를 출력함
