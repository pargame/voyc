# VS Code 단축키 노트: 터미널과 탭

이 문서는 VS Code에서 자주 쓰는 터미널 관련 기능과 탭 관련 기능을 정리한 노트입니다.

## 1. 터미널 관련 기능

- 터미널 보이기/숨기기 — `⌃` + `` ` `` (환경 설정에 따라 다를 수 있음)
- 새 터미널 생성 — `⌘` + `⇧` + `` ` ``
- 임베디드 터미널 간 포커스 이동 — `⌥` + `⌘` + `→` / `⌥` + `⌘` + `←`

## 2. 사이드바 관련 기능

- Explorer 열기 — `⌘` + `⇧` + `E`
- Search 열기 — `⌘` + `⇧` + `F`
- Source Control 열기 — `⌘` + `⇧` + `G`
- Run/Debug 열기 — `⌘` + `⇧` + `D`
- Extensions 열기 — `⌘` + `⇧` + `X`
- 사이드바 토글 — `⌘` + `B`

## 3. 탭 관련 기능

- 열린 탭 이동 - `⌃` + `Tab` (탭 순환) 또는 `⌘` + `⌥` + `→`/`←`
- 현재 탭 닫기 - `⌘` + `W`
- 현재 외 탭 모두 닫기 - `⌥` + `⌘` + `T`
- 모든 탭 닫기 - `⌘` + `K` then `W`

## 4. 편집기 관련 기능

 - `⌘` + `⇧` + `V` — 마크다운 미리보기 열기
 - `⌘` + `K` then `V` — 마크다운 미리보기를 옆에 열기
