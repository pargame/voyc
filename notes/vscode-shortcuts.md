# VS Code 단축키

자주 쓰는 터미널과 탭 단축키를 정리합니다.

## 터미널

| 동작 | 단축키 |
|------|--------|
| 터미널 보이기/숨기기 | `` Ctrl+` `` |
| 새 터미널 | `Cmd+Shift+` |
| 터미널 간 이동 | `Alt+Cmd+→/←` |

## 사이드바

| 동작 | 단축키 |
|------|--------|
| Explorer | `Cmd+Shift+E` |
| Search | `Cmd+Shift+F` |
| Source Control | `Cmd+Shift+G` |
| 사이드바 토글 | `Cmd+B` |
| 커맨드 팔레트 | `Cmd+Shift+P` |

## 탭

| 동작 | 단축키 |
|------|--------|
| 탭 순환 | `Ctrl+Tab` |
| 현재 탭 닫기 | `Cmd+W` |
| 다른 탭 모두 닫기 | `Alt+Cmd+T` |
