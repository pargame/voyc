# zsh 기본 노트

이 노트는 macOS에서 기본적으로 사용하는 `zsh` 셸의 핵심 기능과, 터미널을 재시작하는 방법, 입력 중인 명령을 한꺼번에 삭제하는 방법을 정리합니다.

## 1. zsh란?

- `zsh`는 macOS 기본 셸이자, Bash보다 확장된 기능을 가진 쉘입니다.
- 명령 줄 자동 완성, 히스토리 검색, 플러그인 프레임워크 등이 강점입니다.
- 주로 설정 파일로 `~/.zshrc`를 사용합니다.

## 2. 기본 명령어

- `cd <dir>`: 디렉터리 이동
- `ls`: 현재 디렉터리 목록 표시
- `pwd`: 현재 작업 디렉터리 출력
- `echo <text>`: 문자열 출력
- `clear` 또는 `Ctrl+L`: 화면 지우기

## 3. 셸 설정 파일

- `~/.zshrc`는 zsh가 시작할 때 읽는 사용자 설정 파일입니다.
- 환경 변수, alias, PATH 추가, 프롬프트 설정 등을 여기에 작성합니다.
- 변경 후에는 파일을 다시 읽어야 설정이 적용됩니다.

## 4. 터미널 재시작 및 설정 반영

### 4.1. 터미널 재시작

- macOS 터미널이나 VS Code 통합 터미널을 완전히 닫았다가 다시 엽니다.
- VS Code에서는 통합 터미널을 종료한 뒤 새 터미널을 열어야 `~/.zshrc` 변경이 반영됩니다.

### 4.2. 설정 파일 다시 읽기

- 전체 터미널을 다시 시작하고 싶지 않을 때는 다음 명령을 실행합니다.

```bash
source ~/.zshrc
```

- 또는 같은 효과를 내는 명령:

```bash
. ~/.zshrc
```

### 4.3. 셸 자체 재실행

- 현재 세션에서 zsh를 다시 시작하려면:

```bash
exec zsh
```

- 이 명령은 현재 쉘 프로세스를 새 zsh 프로세스로 대체합니다.

## 5. 입력 중인 명령 전체 삭제

### 5.1. 현재 줄 전체 삭제

- `Ctrl+U`: 커서 위치 앞의 전체 텍스트를 삭제합니다.
- 명령을 작성하던 중 전체를 지우고 싶을 때 가장 많이 사용하는 단축키입니다.

### 5.2. 커서 뒤의 텍스트 삭제

- `Ctrl+K`: 커서부터 줄 끝까지 삭제합니다.

### 5.3. 마지막 단어 삭제

- `Ctrl+W`: 커서 앞의 마지막 단어를 삭제합니다.

### 5.4. 전체 명령어 취소

- `Ctrl+C`: 현재 입력 중인 명령을 취소하고 새 프롬프트로 돌아갑니다.
- 이 방법은 입력 중인 명령을 완전히 포기할 때 사용합니다.

## 6. 히스토리와 검색

- `↑` / `↓` 화살표: 이전/다음 명령어 탐색
- `Ctrl+R`: 명령 기록을 실시간 검색
- `history`: 명령어 기록 출력

## 7. 자주 사용하는 zsh 단축키

- `Ctrl+A`: 줄 처음으로 이동
- `Ctrl+E`: 줄 끝으로 이동
- `Alt+B`: 이전 단어로 이동
- `Alt+F`: 다음 단어로 이동
- `Ctrl+L`: 화면 지우기

## 8. 요약

- `zsh` 설정 변경 후 `source ~/.zshrc` 또는 터미널 재시작으로 반영합니다.
- 입력 중인 명령 전체 삭제는 `Ctrl+U`, 취소는 `Ctrl+C`를 사용합니다.
- `Ctrl+W`, `Ctrl+K` 등으로 부분 삭제도 가능합니다.
- 히스토리 검색과 단축키를 활용하면 셸 사용이 더 빠릅니다.
